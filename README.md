# HTTP status codes.
 * HTTP response status codes indicate whether a specific HTTP request has been successfully completed. Responses are grouped in five classes: 

1. Informational responses 1XX (100 – 199)
2. Successful responses 2XX (200 – 299)
3. Redirection messages 3XX (300 – 399)
4. Client error responses 4XX (400 – 499)
5. Server error responses 5XX (500 – 599)

HTTP status codes.
